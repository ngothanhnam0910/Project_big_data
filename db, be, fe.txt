Note: Tên symbol: viết hoa. Code backend, frontend viết theo model trong cassandra/create_model.py
---------------------------------------------------
1. Cài đặt môi trường
Cài Java bản SE Runtime Environment 8u251
https://www.oracle.com/java/technologies/javase/javase8u211-later-archive-downloads.html

Cài cassandra bản Apache Cassandra 3.11.16
https://cassandra.apache.org/_/download.html

Thêm biến môi trường cho java, cassandra, python (py3)
https://phoenixnap.com/kb/install-cassandra-on-windows

Cài Nodejs

Cài cassandra driver: Pip install cassandra-driver

2. Chạy cassandra:
Start cassandra:
https://phoenixnap.com/kb/install-cassandra-on-windows

Chạy cassandra/create_model.py

3. Chạy server:
chạy server/app.py

4. Chạy front-end:
cd ui/ui_big_data
npm install
npm run dev
